# Repository Rest service MONGODB

Чтобы запустить сервис, необходимо выполнить:

```
docker-compose up -d
```

Затем перейти в браузере по адресу:

```
http://localhost:8080
```

Чтобы посмотреть логи выполнить команду:

```
docker-compose logs -f
```
